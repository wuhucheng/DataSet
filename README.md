#### 4、NWPU VHR-10
> 数据集介绍
> 
>   **数据集背景：**
>   
>      
    NWPU VHR-10数据集,包含10种类型的遥感地物目标,提取自Google Earth、Vaihingen,由西北工业大学于2014年发布,采用horizontal bounding boxes (HBB)标注格式
